Image Comparer

About

This component is not a material design component, but i need to use this in one of my personal projects, then i decided to create this component (this first version code is based on the MetroUI component), then i make it. This is licenced under the MIT Licence. See in Github.

Component Information

The image comparer component is created using the html notation 
<div class="image-compare">
    <img src="./images/comparer-1.jpg">
    <img src="./images/comparer-2.jpg">
</div>
And the component is initialized by the JavaScript code 
var el = document.querySelector('.image-compare');
var comp = new ImageCompare(el, options);

The component options

  • width
  • height
  • sliderClass
  • sliderContent
  • onResize (event)
  • onSliderMove (event)

Setting options via data-*

The initialization options can be setted directly in the html code in the image-compare div: 
<div class="image-compare" data-width="50%">
    <img src="./images/comparer-1.jpg">
    <img src="./images/comparer-2.jpg">
</div>

The component size

The component size can be set by the width and height property in options. The width and height properties accept number, percent or the aspect ratio for the height. Some examples of use this properties is below:

Fixed Width and Height

<div class="image-compare" data-width="300" data-height="300">
    <img src="./images/comparer-1.jpg">
    <img src="./images/comparer-2.jpg">
</div>

Fixed Width and Proportional Height

<div class="image-compare" data-width="300" data-height="auto">
    <img src="./images/comparer-1.jpg">
    <img src="./images/comparer-2.jpg">
</div>
<div class="image-compare" data-width="300" data-height="4/3">
    <img src="./images/comparer-1.jpg">
    <img src="./images/comparer-2.jpg">
</div>

Relative Width and Proportional Height

<div class="image-compare" data-width="70%" data-height="4/3">
    <img src="./images/comparer-1.jpg">
    <img src="./images/comparer-2.jpg">
</div>

sliderClass

The sliderClass property in options is used to set classes to the slider in the image comparer. This property accepts string or array, if this is a string, this is added to classList property of the slider, but if is a array, each element in the array is added to classList of the slider. You can use this to set the color or format, like a circle, in the slider, see some examples: 
var el = document.getElementById('image-compare-option-red-circle');
var comp = new ImageCompare(el, {
    sliderClass: ['red', 'circle']
});

sliderContent

The sliderContent property in options is used to set the content of the slider, for example if you need to add an icon in the slider. See an example: 
var el = document.getElementById('image-compare-slider-content');
var comp = new ImageCompare(el, {
    width: '70%',
    sliderClass: ['red', 'circle', 'icon-to-center', 'noselect'],
    sliderContent: '<i class="material-icons">keyboard_tab</i>',
});
Observation
In the example above, is used a class icon-to-center to centralize the icon and noselect to disable the selection in the slider, for a best use of the icon. The style of the first class is just a flex display with align-items and justify-content to center, and the style of the second class is set the variants of user-select to none (taken from stackoverflow), see: 
.icon-to-center {
    display: flex;
    align-items: center;
    justify-content: center;
}
.noselect {
    -webkit-touch-callout: none; /* iOS Safari */
      -webkit-user-select: none; /* Safari */
       -khtml-user-select: none; /* Konqueror HTML */
         -moz-user-select: none; /* Old versions of Firefox */
          -ms-user-select: none; /* Internet Explorer/Edge */
              user-select: none; /* Non-prefixed version, currently
                                    supported by Chrome, Opera and Firefox */
}

onResize

The onResize event has the arguments (width, height, element). 
var el = document.querySelector('image-compare');
var comp = new ImageCompare(el, {
    width: '70%',
    onResize: function(w, h, element) {
        console.log("resized");
    }
});

onSliderMove

The onSliderMove event has the arguments (x, left_slider_pos, slider_element). This event is raised when the slider is moved. 
var el = document.querySelector('image-compare');
var comp = new ImageCompare(el, {
    width: '100%',
    onSliderMove: function(x, pos, slider) {
        console.log("moved");
    }
});
